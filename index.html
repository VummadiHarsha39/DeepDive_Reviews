Workflow: User inputs review → FastAPI backend processes text → Emotion Classifier generates 28-emotion probabilities → Deception Detector fuses embeddings + emotions to detect authenticity → Explanation Generator creates human-readable verdict + intent summary → Results logged in MLflow

Project information

Technologies:
Hugging Face Transformers, PyTorch, MLflow, FastAPI, Next.js, React, Tailwind CSS, GoEmotions Dataset
Languages:
Python, JavaScript, TypeScript
Source code:
GitHub Repository
Duration:
May 2025 - July 2025
  • Emotion Classifier: Fine-tuned BERT model performing multi-label classification across 28 emotions, interpreting reviews beyond simple sentiment analysis to capture rich emotional states.
  • Deception Detector: Custom PyTorch Transformer model that fuses raw text embeddings and emotion probabilities to determine if a review is real or fake.
  • Explanation Generator: Produces human-readable explanations for the authenticity verdict and summarizes the reviewer's intent using emotion entropy and dominance heuristics.
  • End-to-End Integration: MLflow for experiment tracking and model versioning; FastAPI serves predictions as web endpoints; Next.js frontend provides an interactive interface for users.
